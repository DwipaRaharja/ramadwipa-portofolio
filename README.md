# Ramadwipa Portfolio

Personal portfolio website for Ramadwipa, a full-stack developer. It showcases experience, technology skills, selected projects, testimonials, and contact information.

## Tech stack

- React 19 and TypeScript
- Vite
- Tailwind CSS 4
- Motion
- Phosphor Icons and React Simple Icons

## Development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```
